# Urban Street UI Design

A fictional streetwear brand login and registration page design built with HTML and Tailwind CSS.

## Overview
Urban Street is a front-end UI design project featuring modern login and registration pages for a fictional streetwear clothing brand. The project focuses on clean design, responsive layout, and smooth user experience.

## Features
- Responsive login page
- Responsive registration page
- Dark modern streetwear-inspired design
- Navigation between login and register pages
- Styled form inputs and buttons
- Mobile-friendly layout

## Built With
- HTML
- Tailwind CSS

## Notes
This project is a front-end UI design concept only and does not include backend authentication or database functionality.

## Purpose
Created as a practice project to improve front-end development and responsive UI design skills.
